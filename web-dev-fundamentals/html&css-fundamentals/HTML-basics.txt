Basic HTML :

<!DOCTYPE html>: Specifies the HTML version.
<html>: The root element of an HTML document.
<head>: Contains metadata not visible on the web.
<title>: Sets the title of the web page (seen on the browser tab).
<body>: Contains content visible on the web.
<p>: Paragraph tag used to define a paragraph.
Content: "Hello World!"
Hypertext Markup Language (HTML):

Tags: Anything inside "<" and ">" is not displayed in the browser.
Elements: Tags come in pairs; some may not have closing tags.
Attributes:
id: Provides a unique identifier for CSS selection.
class: Assigns an element to a CSS or JavaScript class.
title: Adds a hover feature displaying text over an element.
HTML Elements Reference:

<html>: Represents the root element of an HTML document.
<head>: Contains machine-readable information (metadata).
<body>: Represents the content of an HTML document; only one per document.
