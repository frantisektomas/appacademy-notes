HTML basics :

HTML stands for HyperText Markup Language.
It is the standard markup language used to create web pages.
HTML describes the structure of a web page using markup.

HTML Document Structure:
The basic structure of an HTML document includes:

<!DOCTYPE html>
<html>
  <head>
    <title>Your Page Title</title>
  </head>
  <body>
    <!-- Your content goes here -->
  </body>
</html>

Tags:
HTML uses tags to define elements on a webpage.
Tags are enclosed in angle brackets (< >).
Example: <p> for paragraphs, <h1> for headings, <a> for links.

Headings:
Used to define headings on a page.

<h1> to <h6> for different levels of headings.
Paragraphs:
Defined using the <p> tag.

<p>This is a paragraph.</p>

Links:
Created using the <a> (anchor) tag.

<a href="https://www.example.com">Visit Example.com</a>

Images:
Displayed using the <img> tag.

<img src="image.jpg" alt="Description of the image">

Lists:
Ordered list (<ol>) and unordered list (<ul>).
List items are defined with the <li> tag.

Forms:
Used to collect user input.
Form elements include <input>, <select>, <textarea>, etc.

Attributes:
Provide additional information about HTML elements.

<a href="https://www.example.com" target="_blank">Visit Example.com</a>

Comments:
Added using <!-- Comment goes here -->.
Comments are not displayed on the webpage but are helpful for code readability.
