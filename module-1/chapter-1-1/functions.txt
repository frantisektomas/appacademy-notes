FUNCTIONS

a function is a set of code that will run when called.

function for average number : 

function average(number1, number2) { // declared 
    return (number1 + number2) / 2;
}

average(12, 16); // calling a function

ORDER OF CODE

In other words, JavaScript will evaluate your code left-to-right and top-to-bottom.

RETURN A VALUE

We'll start with a statement: Every function in JavaScript returns undefined unless otherwise specified. 
When we hit a return statement, we immediately exit the function, jump back to where we called the function,
and evaluate the function call to the value it returned.

PARAMETERS AND ARGUMENTS

Parameters are comma separated variables specified as part of a function's declaration.
Arguments are values passed to the function when it is invoked.

PROJECTS 

goodbye

function goodbye(name) {
  return "Bye " + name + "."
}

console.log(goodbye("Daniel")); // => "Bye Daniel."
console.log(goodbye("Mark")); // => "Bye Mark."
console.log(goodbye("Beyonce")); // => "Bye Beyonce."

plus 5 

function plusFive(num) {
    return num + 5;
}

console.log(plusFive(10));  // => 15
console.log(plusFive(2));   // => 7
console.log(plusFive(-8));  // => -3

average of two

function averageOfTwo(num1, num2) {
  return (num1 + num2) / 2;
}

console.log(averageOfTwo(3, 7)); // => 5.0
console.log(averageOfTwo(16, 5)); // => 10.5
console.log(averageOfTwo(2, 7.5)); // => 4.75

whisper

function whisper(str) {
    return str.toLowerCase()
}

console.log(whisper("Hey Buddy")); // => "...hey buddy..."
console.log(whisper("YEA! that was fun")); // => "...yea! that was fun..."

average of four

function averageOfFour(num1, num2, num3, num4) {
    return (num1 + num2 + num3 + num4) / 4
}

console.log(averageOfFour(10, 10, 15, 5)); // => 10
console.log(averageOfFour(3, 10, 11, 4)); // => 7
console.log(averageOfFour(1, 2, 3, 4)); // => 2.5

echo

function echo(string) {
    console.log(string.toUpperCase() + " ... " + string + " ... " + string.toLowerCase())
}

echo("Mom!"); // => prints "MOM! ... Mom! ... mom!"
echo("hey"); // => prints "HEY ... hey ... hey"
echo("JUMp"); // => prints "JUMP ... JUMp ... jump"
